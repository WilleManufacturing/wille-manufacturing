# Wille Manufacturing LLC

This repository contains the website for Wille Manufacturing LLC.

## How to use

1. Download all files.
2. Upload them to GitHub repository.
3. Enable GitHub Pages in Settings → Pages → Branch: main.
4. Site will be live at: https://YOURUSERNAME.github.io/wille-manufacturing

## Files

- `index.html` – Main page
- `style.css` – Styling
- `script.js` – JavaScript
- `logo.png` – Company logo
